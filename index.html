Meet R 4.1

  • New ...names() utility
  • type.convert() now warns when its as.is argument is not specified,
  • New functions numToBits() and numToInts()
  • c() now removes NULL arguments before dispatching to methods
  • capture.output() no longer uses non-standard evaluation
  • textConnection() gets an optional name argument.
